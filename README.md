# Viikko 1

## Datamalli

Task-data class sisältää:
- id: tunniste
- title: tehtävän nimi
- description: kuvaus
- priority: prioriteetti
- dueDate: päivämäärä
- done: onko valmis (true/false)

## Funktiot

- addTask(list, task): lisää tehtävän listaan
- toggleDone(list, id): vaihtaa taskin done tilan
- filterByDone(list, done): suodattaa valmiit ja keskeneräiset tehtävät
- sortByDueDate(list): laittaa tehtävät määräpäivän mukaan
